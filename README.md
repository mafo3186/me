# mareikefocken-de
My Homepage / Portfolio

## Tech Stack

- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Static Export for shared hosting

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is configured for static export and can be deployed to any shared hosting provider. See [DEPLOYMENT.md](./DEPLOYMENT.md) for detailed instructions.

The built static files will be in the `out/` directory after running `npm run build`.
